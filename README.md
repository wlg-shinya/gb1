# GB1

## 動作環境

- Windows

## スタートアップ

- app/build_run.bat を実行するとロム(app/output/rom/gb1.gb)が作成されてエミュレータが立ち上がり実行確認できます。

## 外部ツールやライブラリ

| 名称&リンク                                          | 概要                       | OS  | バージョン |
| ---------------------------------------------------- | -------------------------- | --- | ---------- |
| [GBDK-2020](https://github.com/gbdk-2020/gbdk-2020)  | GB 開発キット              | Win | 4.4.0      |
| [BGB](https://bgb.bircd.org/)                        | デバッグ用 GB エミュレータ | Win | 1.6.6      |
| [SameBoy](https://github.com/LIJI32/SameBoy)(winsdl) | 製品用 GB エミュレータ     | Win | 1.0.2      |

## (おまけ)GBDK-2020 のサンプルの動作確認

- lib/gbdk/examples/gb/compile.bat を実行すると各フォルダ内以下にロム(.gb)が作成されますので参考にどうぞ
